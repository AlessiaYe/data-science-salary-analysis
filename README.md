# Data Scientist Salary Analysis

This project presents a statistical analysis of Data Scientist salaries based on a real dataset obtained from Kaggle.

## Objectives
- Analyze salary distribution by experience level
- Study employment type differences
- Explore relationships between job conditions and salary

## Dataset
Data Science Salaries Dataset (3755 observations).

## Methods
- Data cleaning and preprocessing
- Contingency tables
- Boxplots and histograms
- Bar charts and percentage analysis

## Tools
- R
- ggplot2
- dplyr

## Key Insights
- Executive-level professionals show the highest salary distribution.
- Full-time positions dominate across all experience levels.
- Contract-based roles present higher salary concentration.
